使用方法
1.  将原始文件放入 dataset/raw 文件夹中，按照示例命名，
    csv 文件按示例存放睡眠类型标号，
    另外可以在程序中更改读取数据的路径。
2.  根据数据的属性，更改变量
    - sonpy_extract.py 中第 83, 84 行
    - psd.py 中第 120 至 127 行
    更改为适合的值。
2.  直接运行 sonpy_extract.py，该程序将
    - smrx 转换为通用格式
    - 同时校验文件完整性
3.  选择好文件编号后运行 psd.py，该程序将完成
    - 信号滤波
    - 睡眠类型提取
    - 傅里叶变换
    - 计算频段功率占比
    - 总体数据平均
    - 画图


注意：
1.  一些个体在低于 2Hz 的频段，其信号功率异常偏大；
    目前开启了 1.0 Hz 的高通滤波器，可视需要调整或删除。
2.  信号数据进行展开与滤波需要占用大量计算资源与存储资源，对内存与硬盘容量有一定要求；
    在需要处理的个体较多时耗时偏长是正常现象。
3.  在 VsCode 中运行 python 代码，首先需要安装 python 插件；
    然后如果提示没有解释器，则需要先在右下角选择一个 python 环境；
    执行代码请在打开 python 文件后再单击右上方三角形执行按钮。
